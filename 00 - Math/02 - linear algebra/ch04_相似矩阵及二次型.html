                                  • 1. 向量的内积、长度及正交性

                                  1.1 向量的内积、长度

                                  @Definition 定义1.1 设有 n 维向量 x=[x1x2⋮xn], y=[y1y2⋮yn],令

                                  [x,y]=xTy=x1y1+x2y2+⋯+xny+n

                                  称 [x,y] 为向量 x 与 y 的内积

                                  向量内积有如下性质(x,y和z 都是 n 维列向量, λ 为实数):

                                  1. [x,y]=[y,x]
                                  2. [λx,y]=λ[x,y]=[x,λy]
                                  3. [x+y,z]=[x,z]+[y,z]
                                  4. [x,x]≥0 ,当且仅当 x=0 时, [x,x]=0

                                  @Definition 定义1.2 设有 n 为向量 x=[x1x2⋮xn] , 令

                                  ||x||=[x,x]=x12+x22+⋯+xn2

                                  称 ||x|| 为向量 x 的长度或范数

                                  向量的长度具有以下性质:

                                  1. 非负性: 当 x≠0 时, ||x||>0; 当 x=0 时, ||x||=0
                                  2. 齐次性: ||λx||=|λ|⋅||x||
                                  3. 三角不等式: ||x+y||≤||x||+||y||

                                  当 ||x||=1 时,称 x 为单位向量, 如果 α≠0, 取 β=α||α||, 则 β 是一个单位向量,由向量 α 得到单位向量 β 的过程称为把向量 α 单位化

                                  @Definition 定义1.3 x≠0,y≠0 时

                                  θ=arccos⁡[x,y]||x||⋅||y||

                                  称为 n 维向量 x 与 y 的夹角

                                  1.2 正交向量组

                                  @Definition 定义1.4 由一组两两正交的非零向量组成的向量组,称为正交向量组。

                                  正交向量组有如下性质

                                  @Theorem 定理1.1 若 n 维向量组 α1,α2,⋯,αm 是一个正交向量组,则该向量组线性无关。

                                  @Definition 定义1.5 设 n 维向量组 ε1,ε2,⋯,εr 是向量空间 V(V⊆Rn) 的一个基,如果 ε1,ε2,⋯,εr 两两正交,且都是单位向量,则称 ε1,ε2,⋯,εr 是 V 的一个规范正交基

                                  1.3 施密特正交化过程

                                  设 α1,α2,⋯,αm 是向量空间 V 的一个基, 从基 α1,α2,⋯,αm 出发, 找一组两两正交的单位向量 ε1,ε2,⋯,εr, 使 ε1,ε2,⋯,εr 与 α1,α2,⋯,αm 等价,这个过程称为把基 α1,α2,⋯,αm 规范正交化。其步骤如下：

                                  1. 将基 α1,α2,⋯,αm 正交化,即：

                                    β1=α1β2=α2−[β1,α2][β1,β1]β1β3=α3−[α1,α3][β1,β1]β1−[β2,β3][β2,β2]β2⋯⋯βr=αr−[β1,αr][β1,β1]β1−[β2,α2][β2,β2]−⋯−[βr−1,αr][βr−1,βr−1]βr−1

                                  2. 将 β1,β2,⋅,βr 单位化,得到

                                    ε1=1||β1||β1,ε2=1||β2||β2,⋯,εr=1||βr||βr

                                    于是, ε1,ε2,⋯,εr 是 V 的一个规范正交基

                                  4. 正交矩阵

                                  @Definition 定义1.6 如果 n 阶矩阵 A 满足

                                  ATA=E(即A−1=AT)

                                  那么称 A 为正交矩阵,简称正交阵。

                                  @Theorem 定理1.2 设矩阵 A 是 n 阶方阵,则下列结论等价:

                                  1. A 是 n 阶正交阵
                                  2. A 的列向量组是 Rn 的一个规范正交基
                                  3. A 的行向量组是 Rn 的一个规范正交基

                                  @Definition 定义1.7 若 P 为正交矩阵,则线性变换 y=Px 称为正交变换 设 y=Px 为正交变换,则有：

                                  ||y||=yTy=xTPTPx=||x||

                                  2. 方阵的特征值与特征向量

                                  2.1 方阵的特征值与特征向量的概念及其求法

                                  @Definition 定义2.1 设 A 是 n 阶矩阵,如果数 λ 和 n 维非零向量 α 使关系式

                                  Aα=λα

                                  成立,那么数 λ 称为矩阵 A 的特征值, 非零向量 α 称为 α 的对应于特征值 λ 的特征向量。

                                  假设矩阵 A 有特征值 λ ,对应于特征值 λ 的特征向量为 λ , 则有:

                                  (A−λE)α=0

                                  而方程组有非零解的充分必要条件是系数的行列式等于零,即

                                  |(A−λE)α|=0

                                  记:

                                  f(λ)=|A−λE|=|a11−λa12⋯a1na21a22−λ⋯a2n⋮⋮⋱⋮an1an2⋯ann−λ|

                                  则 f(λ) 是 λ 的 n 次多项式，称为矩阵 A 的特征多项式。 从而 |A−λE|=0 可写成 f(λ)=0, 这是以 λ 为未知数的一元 n 次方程, 称为 A 的特征方程, 而 A 的特征值就是特征方程的根。

                                  设 λ=λi 是矩阵 A 的一个特征值, 则由方程

                                  (A−λiE)x=0

                                  可求得非零解 x=αi , 那么 αi 就是 A 对应于特征值 λi 的特征向量。

                                  2.2 方阵的特征值与特征向量的性质

                                  @Property 性质2.1 设 n 阶方阵 A=(aij) 的特征值为 λ1,λ2,⋯,λn, 则

                                  1. λ1+λ2+⋯+λn=a11+a22+⋯+ann
                                  2. λ1λ2⋯λn=|A|

                                  @Property 性质2.2 若 λ 是方阵 A 的特征值, α 是对应于特征值的特征向量, 则

                                  1. λk 是方阵 Ak 的特征值(k 为非负整数), 对应于特征值 λk 的特征向量是 α
                                  2. kλ 是方阵 kA 的特征值( k 为任意常数), 对应于特征值 kλ 的特征向量是 α
                                  3. 当 kA 可逆时, λ−1 是方阵 A−1 的特征值,对应于特征值 kλ−1 的特征向量是 α
                                  4. 若矩阵 A 的多项式是 φ(A)=amAm+⋯+a1A+a0E , 则方阵 φ(A) 的特征值是 φ(λ), 对应特征值 φ(λ) 的特征值是 α

                                  @Property 性质2.3 如果 α1 与 α2 是方阵的同一特征值 λ 所对应的特征向量,则 k1α1+k2α2 ( k1、k2 不同时为零) 也是特征值 λ 所对应的特征向量

                                  @Propety 性质2.4 设 λ1,λ2,⋯,λm 是方阵 A 的 m 个互不相同的特征值, α1,α2,⋯,αm 是依次与之对应的特征向量, 则 α1,α2,⋯,αm 线性无关

                                  @Property 性质2.5 设 λ1 和 λ2 是矩阵 A 的两个不同的特征值, α1,α2,⋯,αs 和 β1,β2,⋯,βt 是分别对应于 λ1 和 λ2 的线性无关的特征向量, 则 α1,α2,⋯,αs,β1,β2,⋯,βt 线性无关

                                  3. 相似矩阵

                                  3.1 方阵的相似矩阵的定义和性质

                                  @Definition 定义3.1 设 A,B 是 n 阶矩阵,若有可逆矩阵 P, 使

                                  P−1AP=B

                                  则称 B 是 A 的相似矩阵, 或者说矩阵 A 与 B 相似, 对 A 进行运算 P−1AP 称为对 A 进行相似变换, 可逆矩阵 P 称为把 A 变成 B 的相似变换矩阵

                                  @Thoerem 定理3.1 若 n 阶矩阵 A 与 B 相似, 则 A 与 B 有相同的特征多项式,从而 A 与 B 有相同的特征值

                                  3.2 方阵的相似对角化

                                  @Theorem 定理3.2 n 阶矩阵 A 与对角阵相似( A 能对角化) 的充分必要条件是 A 有 n 个线性无关的特征向量

                                  由上,有如下结论: 如果 n 阶矩阵 A 的 n 个特征值互不相等, 则 A 与对角阵相似

                                  4. 实对称矩阵的相似对角化

                                  4.1 实对称矩阵的特征值和特征向量的性质

                                  @Property 性质4.1 实对称矩阵的特征值为实数

                                  @Property 性质4.2 设 λ1,λ2 是实对称矩阵 A 的两个特征值, p1,p2 是对应的两个特征向量,若 λ1≠λ2, 则 p1 与 p2 正交

                                  4.2 实对称矩阵的相似对角化

                                  @Theorem 定理4.1 n 阶实对称矩阵 A 必定正交于相似对角阵 Λ, 即存在正交阵 P, 使得 P−1AP=PTAP=Λ, 其中 Λ 是对角线上的元素是 A 的 n 个特征值

                                  由上述定理可得： 设 A 为 n 阶实对称矩阵, λ 是 A 的特征方程的 k 重根, 则矩阵 A−λE 的秩 R(A−λE)=n−r, 从而特征值 λ 有 k 个线性无关的特征向量

                                  下面是将实对称矩阵 A 对角化的步骤:

                                  1. 求出 A 的全部互不相等的特征值 λ1,λ2,⋯,λn, 它们的重数依次为 k1,k2,⋯,ks(k1+k2+⋯+ks=n)
                                  2. 对于每个 ki 重特征值 λi , 求方程 (A−λiE)x=0 的基础解系,得 ki 个线性无关的特征向量, 再把它们正交化、单位化,得 ki 个两两正交的单位特征向量。因 k1+k2+⋯+ks=n, 故总共可得 n 个两两正交的单位特征向量
                                  3. 把这 n 个两两正交的单位特征向量构成正交阵 P, 便有 P−1AP=PTAP=Λ

                                  5. 二次型及其标准形

                                  5.1 二次型及其标准形的定义

                                  @Definition 定义5.1 含有 n 个变量 x1,x2,⋯,xn 的二次齐次多项式

                                  f(x1,x2,⋯,xn)=a11x12+2a12x1x2+2a13x1x3+⋯+2a1,n−1x1xn−1+2a1nx1xn+a22x22+2a23x2x3+⋯+2a2,n−1x2xn−1+2a2nx2xn+⋯+an−1,n−1xn−1xn−1+2an−1,nxn−1xn+an,nxnxn

                                  称为二次型, 如果所有系数 aij(1≤i,j≤n) 均为实数, 则称二次型为实二次型。 如果 n 元二次型 f(x1,x2,⋯,xn) 只含有平方项,即

                                  f(x1,x2,⋯,xn)=k1x12+k2x⋯2knxn2

                                  称这样的二次型为二次型的标准型。 如果标准型的系数 k1,k2,⋯,kn 只在 1, -1, 0 三个数中取值,也就是

                                  f(x1,x2,⋯,xn)=x12+⋯+xp2−xp+12−⋯−xr2

                                  就称其为二次型的规范形

                                  任给一个二次型,就唯一确定一个对称阵; 反之一个对称阵也可唯一确定一个二次型。我们把对称阵 A 叫做二次型 f(x)=xTAx 的矩阵,也把 f(x)=xTAx 叫做对称阵 A 的二次型。

                                  5.2 用正交变换化二次型为标准型

                                  @ Definition 定义5.2 设 A 和 B 是 n 阶矩阵, 若有可逆矩阵 C , 使 B=CTAC, 则称矩阵 A 与 B 合同。 合同关系时一个等价关系,即满足

                                  1. 反身性: 每个一方阵都与它自身合同
                                  2. 对称性：如果 A 与 B 合同, 则 B 也与 A 合同。
                                  3. 传递性：如果 A 与 B 合同, B 与 C 合同, 则 A 与 C 合同

                                  @Theorem 定理5.1 任给二次型 f=∑i,j=1naijxixj(aij=aji), 总有正交变换 x=Py, 使 f 化为标准型

                                  λ1y12+λ2y22+⋯+λnyn2

                                  其中 λ1,λ2,⋯,λn 是 f 的矩阵 A=(aij) 的特征值

                                  上述定理可得如下推论: 任给 n 元二次型 f(x)=xTAx(AT=A) ,总有可逆变换 x=Cz , 使 f(Cz) 为规范型

                                  6. 正定二次型与正定矩阵

                                  6.1 惯性定理

                                  @Theorem 定理6.1(惯性定理) 设有二次型 f=xTAx, 它的秩为 r, 有两个可逆变换

                                  x=Cy以及x=Pz

                                  使得

                                  f=k1y12+k2y22+⋯+kryr2(ki≠0)

                                  及

                                  f=λ1z12+λ2z22+⋯+λrzr2(ki≠0)

                                  则 k1,k2,⋯,kr 中正数的个数与 λ1,λ2,⋯,λr 中正数的个数相等

                                  6.2 正定二次型与正定阵

                                  二次型的标准型中正系数的个数称为二次型的正惯性指数,负系数的个数称为二次型的负惯性指数。

                                  @Definition 定义6.1 设有二次型 f=xTAx ,如果对任何 x≠0, 都有 f(x)>0, 则称二次型 f 为正定型二次型,并称对称阵 A 是正定的;如果对任何 x≠0, 都有 f(x)<0, 则称二次型 f 为正定型二次型,并称对称阵 A 是负定的

                                  @Theorem 定理6.2 n 元二次型 f=xTAx 为正定的充分必要条件是它的正惯性指数等于 n , 即它的规范形的 n 个系数全为1

                                  上述定理可得如下推论:

                                  1. 对称阵 A 为正定的充分必要条件是: A 与单位矩阵 E 合同
                                  2. 对称阵 A 为正定的充分必要条件是: A 的特征值全为正

                                  @Theorem 定理6.3(赫尔维茨定理) 对称阵 A 为正定的充分必要条件是: A 的各阶顺序主子式都为正,即

                                  [a11⋯a1r⋮⋱⋮ar1⋯arr]>0,(r=1,2,⋯,n)

                                  为负定的充分必要条件是: 奇数阶顺序主子式都为负,偶数阶顺序子主式为正,即

                                  (−1)r[a11⋯a1r⋮⋱⋮ar1⋯arr]>0,(r=1,2,⋯,n)
