                          1.1 排列

                          1.1.1 排列及其逆序数

                          @Definition 定义1.1 从 1,2,⋯,n 中任意选取 r 个不同的数排成一列,称为排列

                          @Definition 定义1.2 将 1,2,⋯,n 这 n 个不同的数排成一列,称为 n 阶全排列,也简称全排列。

                          @Definition 定义1.3 在一个排列中,如果一对数的排列顺序与自然顺序相反,即排在左边的数比排在它右边的数大,那么它们就称为一位逆序,一个排列中逆序数的总数就称为这个排列的逆序数。排列 i1,i2,⋯,in 的逆序数记为 τ(i1i2⋯in)

                          奇排列与偶排列

                          @Definition 定义1.4 逆序数为偶数的排列,称为偶排列;逆序数为奇数的排列,称为奇排列

                          @Definition 定义1.5 只交换排列中某两个数的位置,其他的数保持不动而得到的一个新排列的变换,称为一个对换。若交换的是相邻位置的两个元素,则称该对换为相邻对换

                          对于排列,有如下结论:

                          @Theorem 定理1.1 对换改变排列的奇偶性

                          @Theorem 定理1.2 在 n 阶排列中,偶排列和奇排列各占一半,即各有 n!2 个

                          1.2 n 阶行列式

                          1.2.1 n 阶行列式的定义

                          有 n2 个元素 aij(i,j=1,2,⋯,n)排成 n 行 n 列的正方形的数表:

                          a11a12⋯a1na21a22⋯a2n⋮⋮⋱⋮an1an2⋯ann

                          由这个数表所决定的数

                          ∑p1p2⋯pn(−1)τ(p1p2⋯pn)a1p1a2p2⋯anpn

                          称为由 n2 个元素 aij(i,j=1,2,⋯,n) 构成的 n 阶行列式,记为

                          Dn=|a11a12⋯a1na21a22⋯a2n⋮⋮⋱⋮an1an2⋯ann|

                          记矩阵

                          A=[a11a12⋯a1na21a22⋯a2n⋮⋮⋱⋮an1an2⋯ann]

                          则行列式通常也称为方阵 A 的行列式,记为 |A|。有时为了表明行列式是由元素 aij 构成的,也简记为 |A|=det(aij)、|aij|n×n 或 |aij|n

                          n 阶行列式具有如下特点:

                          1. ∑p1p2⋯pn 是对所有的 n 阶全排列 p1p2⋯pn 求和,所有展开式中共有 n! 项
                          2. 每一项 a1p1a2p2⋯anpn 是取自不同行不同列的 n 个元素的乘积
                          3. 每一项 a1p1a2p2⋯anpn 的行标排成一个标准排列,列标排列 p1p2⋯pn 的奇偶性决定了乘积 a1p1a2p2⋯anpn 前的符号

                          1.2.2 2阶行列式和3阶行列式

                          当 n=2 时, 由方阵

                          [a11a12⋯a1na21a22⋯a2n⋮⋮⋱⋮an1an2⋯ann]

                          所确定的二阶行列式为:

                          |a11a12⋯a1na21a22⋯a2n⋮⋮⋱⋮an1an2⋯ann|=∑p1p2(−1)τ(p1p2)a1p1a2p2=a11a22−a12a21

                          当 n=3 时,3阶方阵

                          A=[a11a12a13a21a22a23a31a32an3]=∑p1p2(−1)τ(p1p2p3)a1p1a2p2a3p3=a11a22a33+a13a21a32+a12a23a31−a13a21a33−a11a23a32

                          1.3 几类特殊的 n 阶行列式的值

                          无论上三角矩阵还是下三角矩阵,其行列式的值都等于其主对角线上 n 个元素的乘积,而与其他位置的非零元素没有关系。

                           

                          2. 行列式的性质

                          2.1 行列式的性质

                          @Definition 定义2.1 将行列式

                          Dn=|a11a12⋯a1na21a22⋯a2n⋮⋮⋱⋮an1an2⋯ann|

                          各行元素换为同序号的列元素,所得到的行列式

                          DnT=|a11a21⋯an1a12a22⋯an2⋮⋮⋱⋮a1na2n⋯ann|

                          称为行列式 Dn 的转置行列式

                          @Property 性质1 行列式 Dn 的转置与它的转置行列式 DnT 相等

                          @Property 性质2 互换行列式的两行(或两列),行列式变号 若行列式置空有两行(或两列) 对应元素相等,则行列式等于零

                          @Property 性质3 若行列式的某一行(或列) 有公因子 k, 则公因子 k 可以提到行列式记号外面;或者说,用 k 乘以行列式的某一行(或某一列), 等于用 k 乘以该行列式

                          @Theorem 定理2.1 设 A 是 n 阶方阵,则等式 |kA|=kn|A| 成立 由此有如下推论:

                          1. 若行列式的某一行(或某一列) 元素全为零,则行列式的值为零
                          2. 若行列式某两行(或两列) 元素对应成比例,则行列式为零

                          @Property 性质4 有如下的行列式拆分:

                          |a11a12⋯a1n⋮⋮⋱⋮bk1+ck1bk2+ck2⋯bkn+ckn⋮⋮⋱⋮an1an2⋯ann|=|a11a12⋯a1n⋮⋮⋱⋮bk1bk2⋯bkn⋮⋮⋱⋮an1an2⋯ann|=|a11a12⋯a1n⋮⋮⋱⋮ck1ck2⋯ckn⋮⋮⋱⋮an1an2⋯ann|

                          @Property 性质5 行列式某一行(或某一列) 的 k 倍加到另一行(或另一列) 的对应元素上去,行列式的值不变。

                          2.2 行列式可逆的充要条件

                          @Theorem 定理2.2 n 阶方阵 A 可逆的充要条件是 |A|≠0

                          @Theorem 定理2.3 设 A、B 是两个 n 阶方阵,则 |AB|=|A|⋅|B|

                          由上面两个定理,可推得： 设 A 是 n 阶方阵,如果存在 n 阶方阵 B 满足 AB=E(或BA=E) , 则 n 阶方阵 A 可逆,且 A−1=B

                          3. 行列式按行(列) 展开

                          3.1 余子式和代数余子式

                          设 |A| 是 n 阶方阵 A=(aij)m×n 的行列式。对任意的 1≤i,j≤n , 在 |A| 中划去第 i 行和第 j 列后剩下的 n−1 阶行列式称为 (i,j) 元素 aij 的余子式,记为

                          Aij=(−1)i+jMij

                          Aij 称为 (i,j) 元素 aij 的代数余子式。

                           

                          3.2 行列式按行(列) 展开

                          @Theorem 定理3.1 设行列式

                          |A|=|a11a12⋯a1na21a22⋯a2n⋮⋮⋱⋮an1an2⋯ann|

                          则有:

                          |A|=ai1Ai1+ai2Ai2+⋯+ainAin=∑k=1naikAik(i=1,2,⋯,n)

                          和

                          |A|=a1jA1j+a2jA2j+⋯+anjAnj=∑k=1nakjAkj(j=1,2,⋯,n)

                          上面两个式子分别称为 |A| 按第 i 行展开的展开式及按第 j 列展开的展开式。

                          该定理可得如下推论: 设 Aij(i,j=1,2,⋯,n) 是行列式 |A| 中元素 aij 的代数余子式,则:

                          ai1Aj1+ai2Aj2+⋯+ainAjn=0,i≠j

                          或

                          a1iA1j+a2iA2j+⋯+aniAnj=0,i≠j

                          综上,代数余子式的性质:

                          ∑k=1naikAjk=|A|δij={|A|,i=j0,i≠j

                          其中 δij={|A|,i=j0,i≠j 是克罗内克符号。

                          4. 矩阵求逆公式与克莱默法则

                          4.1 伴随矩阵与矩阵的求逆公式

                          @Definition 定义4.1 设 A=(aij) 是 n 阶方阵, Aij 是 |A| 的 (i,j) 元素 aij 的代数余子式,则矩阵

                          A∗|A11A12⋯A1nA21A22⋯A2n⋮⋮⋱⋮An1An2⋯Ann|

                          称为矩阵 A 的伴随矩阵

                          @Theorem 引理 设方阵 A∗ 是 n 阶方阵 A 的伴随矩阵,则有

                          AA∗=A∗A=||A||A|⋱|A||=|A|E

                          @Theorem 定理4.1 如果 n 阶方阵 A 可逆,则有求逆公式 A−1=1|A|A∗

                          4.2 克莱默法则

                          @Theorem 定理4.2(克莱默法则) 如果线性方程组 Ax=β 的系数的行列式不等于零, 即 |A|≠0, 则方程组有唯一解：

                          x1=D1|A|,x2=D2|A|,⋯,xn=Dn|A|

                          其中, Dj(j=1,2,⋯,n) 是把系数行列式的第 j 列元素用 β 替换后得到的行列式。

                          从克莱默法则,可由如下结论：

                          @Theorem 定理4.3 如果线性方程组 Ax=β 的系数行列式不等于零,即 A≠0 , 则方程组一定有解,且解是唯一的。

                          @Theorem 定理4.4 如果线性方程组 Ax=β 无解或有无穷多解,则它的系数行列式必等于零,即 |A|=0

                          @Theorem 定理4.5 如果齐次线性方程组 Ax=0 的系数行列式不等于零,即 |A|≠0, 则它只有零解,即

                          x1=x2=⋯=xn=0

                          @Theorem 定理4.6 如果齐次线性方程组 Ax=0 有非零解,则它的的系数行列式等于零,即 |A|=0
