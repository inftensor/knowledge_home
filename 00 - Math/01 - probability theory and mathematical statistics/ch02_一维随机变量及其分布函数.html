                                          • 1. 随机变量及其分布函数

                                          1.1 随机变量

                                          @Definfiton 定义1.1 设 Ω 为随机试验上的样本空间, X 为 Ω 上的一个单实值函数,若满足对任意的 x∈R , 有 {X≤x}={ω|X(ω)≤x,ω∈Ω}∈F , 其中 F 是事件域, 则称 X 为概率空间 (Ω,F,P) 上的随机变量

                                          1.2 分布函数的定义

                                          @Definition 定义1.2 设 X 是一个随机变量, 记

                                          F(x)=P{X≤x},x∈R

                                          称 F(x) 为 X 的分布函数。 分布函数有以下性质:

                                          1. 非负性 0≤F(x)≤1,x∈R
                                          2. 单调不减性 当 x1<x2 时, 有 F(x1)≤F(x2)
                                          3. 规范性 F(−∞)=limx→−∞F(x)=0, F(+∞)=limx→+∞F(x)=1
                                          4. 右连续性 F(x+0)=limt→x+0F(t)=F(x),x∈R
                                          5. 函数 F(x) 在点 x 处连续的充分必要条件是 P{X=x}=0

                                          2. 离散型随机变量及其分布

                                          2.1 离散型随机变量及其分布律

                                          @Definition 定义2.1 如果随机变量 X 取有限多个或可列多个不同的值,则称 X 为离散型随机变量

                                           

                                          @Definition 定义2.2 设 X 为一个离散型随机变量,它的所有可能取值记为 a1,a2,a3,⋯,, 则称概率 pi=P{X=ai} , i=1,2,3,⋯, 为 X 的分布律或概率分布。 X 的分布律有如下性质:

                                          1. 非负性 pi≥0,i=1,2,3,⋯
                                          2. 正则性 ∑i=1+∞pi=1

                                          2.2 常见离散型随机变量的分布律

                                          2.2.1 退化分布

                                          如果 P{X=c}=1, 其中 c 为常数, 则称 X 服从退化(单点) 分布。退化分布是确定性现象的概率模型。

                                          2.2.2 两点分布

                                          如果随机变量 X 的分布律为:

                                          0<p<1, 则称 X 服从参数 p 的两点分布,特别的,当 a=o,b=1 时, 上表为

                                          0<p<1,则称 X 服从0-1分布,记为 X∼B(1,p) 。 0-1分布的另一种表达形式为

                                          P{X=k}=pk(1−p)1−k,k=0,1

                                          2.2.3 二项分布

                                          如果随机变量 X 的分布律为

                                          P{X=i}=Cnip(1−p)n−i,其中i=0,1,2,⋯,n,0<p<1

                                          则称 X 服从参数为 n 与 p 的二项分布,记为 X∼B(n,p)。

                                          2.2.4 泊松分布

                                          如果随机变量 X 的分布律为

                                          P{X=k}=e−λλkk!,k=0,1,2,⋯,λ>0

                                          则称 X 服从参数为 λ 的泊松分布,记为 X∼P(λ)

                                          @Theorem 定理2.1(泊松定理) 如果 limn→+∞npn=λ>0 ，则

                                          limn→+∞Cnkpnk(1−pn)n−k=e−λλkk!,k=0,1,2,⋯

                                          当 n 很大, p 很小时,参数为 n,p 的二项分布近似与参数为 np 的泊松分布,有如下近似公式

                                          Cnkpnk(1−pn)n−k≈e−λλkk!,k=0,1,2,⋯

                                          2.2.5 几何分布

                                          如果离散型随机变量 X 的分布律为

                                          P{X=k}=p(1−p)k−1,k=1,2,3,⋯,0<p<1

                                          则称 X 服从参数为 p 的几何分布, 记为 X∼G(p)

                                          2.2.6 负二项分布

                                          如果离散型随机变量 X 的分布律为

                                          P{X=k}=Ck−1r−1prqk−r,k=r,r+1,r+2,⋯,0<p<1

                                          则称 X 服从参数为 p 的负二项分布(也称帕斯卡分布)

                                          2.2.7 超几何分布

                                          如果离散型随机变量 X 的分布律为

                                          P{X=k}=CMkCN−Mn−kCNn,k=0,1,⋯,min(n,M)

                                          其中 1≤M≤N,1≤n≤N,n<N−M ,则称 X 服从超几何分布,记为 X∼H(N,M,n)

                                          3. 连续型随机变量及其分布

                                          3.1 连续型随机变量及其密度函数

                                          @Definition 定义3.1 设函数 F(x) 为随机变量 X 的分布函数, 如果 F(x) 可表示为非负函数 f(x) 的积分

                                          F(x)=∫−∞xf(t)dt,x∈R

                                          则称 X 为连续型随机变量。称 f(x) 为 X 的密度函数,简称密度。称 F(x) 为连续型分布函数。 连续型随机变量 X 与分布函数 F(x) 有如下性质

                                          1. 非负性 f(x)≥0,x∈R
                                          2. 规范性 ∫−∞+∞f(x)dx=1
                                          3. P{a<X≤b}=∫abf(x)dx ,其中 a,b 均为常数,且 a<b
                                          4. F(x) 是 x 的连续函数
                                          5. 在 f(x) 的可微点 x 处, 有 F′(x)=f(x)

                                          概率为0的时间不一定是不可能事件,概率为1的时间也不一定是必然事件。

                                          3.2 常见的连续型随机变量的密度函数

                                          3.2.1 均匀分布

                                          如果连续型随机变量 X 的密度函数为

                                          f(x)={1b−a,x∈[a,b]0,x∉[a,b],a<b

                                          则称 X 服从区间 [a,b] 上的均匀分布,记为 X∼U[a,b]

                                          X 的分布函数为

                                          F(x)={0,x<ax−ab−a,a≤x<b1,x≥b

                                          3.2.2 指数分布

                                          如果连续型随机变量 X 的密度函数为

                                          f(x)={λe−λx,x>00,x≤0(λ>0)

                                          则称 X 服从参数为 λ 的指数分布,记为 X∼Γ(1,λ)

                                          @Theorem 定理2.3 设 X 为连续型非负随机变量,则 X 服从指数分布的充分必要条件是对任何 s,t≥0,有

                                          P{X>s+t|X>s}=P{X>t}

                                          称上式为无记忆性或无后效性，它是指数分布的特征。

                                          3.2.3 正态分布

                                          如果连续型随机变量 X 的密度为

                                          f(x)=12πσe−(x−μ)22σ2,x∈R

                                          其中 μ,σ 均为常数, σ>0 ,则称 X 服从参数为 μ 和 σ2 的正态分布,记为 X∼N(μ,σ2)

                                          特别的,当 μ=0 ,且 σ=1 时, 上式变为

                                          f(x)=12πe−x22,x∈R

                                          这时称 X 服从标准正态分布, 即 X∼N(0,1), 并用 ϕ(x) 表示其密度函数, 用 Φ(x) 表示其分布函数,即:

                                          ϕ(x)=12πe−x22,x∈RΦ(x)=∫−∞xϕ(t)dt=∫−∞x12πe−t22dt,x∈R

                                          正态分布函数 N(μ,σ2) 的密度函数 f(x) 、分布函数 F(x) 以及标准正态分布 Φ(x) 具有如下性质：

                                          1. f(x),F(x) 处处为正且存在各阶导数
                                          2. 在 (−∞,μ) 内 f(x) 单调上升, 在 (μ,+∞) 内 f(x) 单调下降, 在 x=μ 处 f(x) 取得最大值 12πσ, 且 limx→∞f(x)=0
                                          3. f(x) 的图形关于直线 x=μ 对称, 即 f(μ+x)=f(μ−x)
                                          4. F(x)=Φ(x−μσ)
                                          5. Φ(−x)=1−Φ(x)

                                          5.1 伽马分布

                                          先介绍伽马函数, 其由积分

                                          Γ(α)=∫0+∞xα−1e−xdx,α>0

                                          定义, 对正数 α 和 正整数 n, 有：

                                          1. Γ(1+α)=αΓ(α)
                                          2. Γ(n)=(n−1)!
                                          3. Γ(12)=π

                                          设 α,β 是正常数,如果 X 的密度函数是

                                          f(x)={βαΓ(α)xα−1e−βx,x≥00,x<0

                                          则称 X 服从参数是 (α,β) 的伽马分布,记为 X∼Γ(α,β)。

                                          5.3 泊松流

                                          源源不断到来的质点,称为流。

                                          @Definition 定义5.1 设某个流在 (0,t] 时段内达到的质点数为 X(0,t]， 记 pk(t)={X(0,t]=k},k∈Z,t≥0,则称 {X(0,t],t≥0}为泊松流,它具有如下特点

                                          1. 平稳性 在事件 (t,t+Δt) 内流过的质点数的概率仅与 Δt 有关, 与 t 无关
                                          2. 无后效性 不相交的时间间隔内流过的质点数彼此独立
                                          3. 普通性 瞬间仅流过1个或0个质点
                                          4. 有限性 在任意有限长的时段内只流过有限多个质点

                                          @Theorem 定理5.1 如果 {X(0,t],t≥0} 是 泊松流, 则都存在某个正数 λ 使得

                                          pk(t)=P{X(0,t]=k}=(λt)kk!e−λt,t=0,1,2,⋯

                                          特别的,当 t=1 时, 上式就是参数为 λ 的泊松分布, 即单位时间内流过的质点数 X(0,1] 服从泊松分布。
